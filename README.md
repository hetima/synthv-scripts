# synthv-scripts
Synthesizer V Studio 用スクリプトです。


## 次のノートと繋げる（Contact Next Note）
- ファイル： ContactNextNote.js
- カテゴリ： ノート編集

選択したノートの長さを変更し、次のノートとの隙間を埋めます。重なっている場合は重なりを除去します。  
複数選択して一括処理可能です。
